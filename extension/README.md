# CodeTracker

CodeTracker is a lightweight developer activity tracking companion for your CodeDash dashboard.

## Features

- **Activity Telemetry**: Records active coding hours, character counts, and modified lines natively in the background.
- **Context Awareness**: Connects workspace folder structures to your active project metrics.
- **Secure Syncing**: Periodically flushes aggregated statistics directly to your local CodeDash server using your api secret.
